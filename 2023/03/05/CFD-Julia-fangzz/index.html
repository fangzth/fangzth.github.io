CFD Julia

从头学CFD!

最近课题组处于项目真空期，自己学习的时间较多，于是下定决心学习收藏已久的 CFD Julia 课程。这门课是俄克拉荷马州立大学的高年级本科生或研究生课程，主要教授CFD的基础，包括：时间和空间的离散、显示与隐式数值格式、多步数值格式、高精度激波捕捉数值方法以及CFD中的迭代求解器。这门课程适用极简的Julia和Python语言，通过解决linear convection equation, inviscid Burgers equation, two-dimensional Poisson equation来说明上述方法。此外，还介绍了不同边界条件下不可压N-S方程的求解（方腔顶盖流和涡流合并问题），最后还对新开发的求解器做了介绍。

这门课是网络上许多CFD专家所推崇的，对于现有教学体系下基础知识薄弱的同学是一个很好的补充。学完不仅可以补充CFD基本知识，还可以提升自己的动手能力和编程能力，能够帮助自己从另外的角度看待工程仿真中出现的问题，再不济也可以熟悉Julia这门语言。

所以，希望在这一个月之内尽快完成这门课的学习。

Julia:便于理解，且计算性能好。

以下是对课程内容解决问题及描述

Index Description
01 1D heat equqtion: Forward time central space (FTCS) scheme
02 1D heat equqtion: Third-order Runge-Kutta (RK3) scheme
03 1D heat equqtion: Crank-Nicolson (CN) scheme
04 1D heat equqtion: Implicit compact Pade (ICP) scheme
05 1D inviscid Burgers equation: WENO-5 with Dirichlet and periodic boundary condition
06 1D inviscid Burgers equation: CRWENO-5 with Dirichlet and periodic boundary condition
07 1D inviscid Burgers equation: Flux-splitting approach with WENO-5
08 1D inviscid Burgers equation: Rieman slover approach with WENO-5 using Rusanov solver
09 1D Euler equations: Roe solver, WENO-5, RK3 for time integration
10 1D Euler equations: HLLC solver, WENO-5, RK3 for time integration
11 1D Euler equations: Rusanov solver, WENO-5, RK3 for time integration
12 2D Poisson equation: Finite difference fast Fourier transform (FFT) based direct solver
13 2D Poisson equation: Spectral fast Fourier transform (FFT) based direct solver
14 2D Poisson equation: Fast sine transform (FST) based direct solver for Dirichlet boundary
15 2D Poisson equation: Gauss-Seidel iterative method
16 2D Poisson equation: Conjugate gradient iterative method
17 2D Poisson equation: V-cycle multi-grid iterative method
18 2D incompressible Navier-Stokes equations (cavity flow): Arakawa, FST, RK3 schemes
19 2D incompressible Navier-Stokes equations (vortex merging): Arakawa, FST, RK3 schemes
20 2D incompressible Navier-Stokes equations (vortex merging): Hybrid RK3/CN approach
21 2D incompressible Navier-Stokes equations (vortex merging): Hybrid pesudo-spectral 3/2 padding approach
22 2D incompressible Navier-Stokes equations (vortex merging): Hybrid pesudo-spectral 2/3 padding approach
